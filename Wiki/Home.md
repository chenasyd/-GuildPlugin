## 📦 模块开发 | Module Development

### 中文
GuildPlugin 采用模块化设计，允许开发者创建自定义模块来扩展插件功能。

**模块示例：**
- 公告模块 (Announcement Module)
- 成员等级模块 (Member Rank Module)
- 公会统计模块 (Guild Stats Module)
- 公会任务模块 (Guild Quest Module)

**开发资源：**
- [SDK 开发者指南](./SDK-Developer-Guide.md) - 学习如何开发模块
- [模块示例配置](../module-example-pom.xml) - Maven 配置示例

### English
GuildPlugin uses a modular design that allows developers to create custom modules to extend plugin functionality.

**Module Examples:**

- Member Rank Module
- Guild Stats Module
- Guild Quest Module
